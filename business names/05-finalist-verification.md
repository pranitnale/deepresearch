## Urjawerk — VERDICT: CLEAR

- **Collisions:** No identical company found. Nearest: URWERK (Swiss haute-horlogerie watches, urwerk.com) — phonetic neighbor, different spelling/sector. The root "Urja" (Sanskrit "energy") is heavily used as an energy-brand root, all India-based: Urja Global, Urja Solutions, URJA-Energy, Urjanet. No identical "Urjawerk".
- **Language:** Clean. "-werk" is a natural, positive German industrial suffix (evokes Kraftwerk — on-theme for energy). "Ur-" reads as "primordial/original", neutral-positive. Fine in EN/FR/ES/IT/NL/PL.
- Cleanest candidate; only caveat is that "Urja" is a common (Indian) energy-brand root, so it is descriptive-generic within the sector.

## Setuwerk — VERDICT: MINOR RISK

Setu.co — well-known Indian fintech API firm ("Setu" = Sanskrit "bridge"), acquired by Pine Labs. No identical "Setuwerk".

- The full compound is distinct but not distinctive; the "Setu" root is taken by a German software GmbH — get legal advice before using this root.

## Krantix — VERDICT: AVOID

- **Collisions:** krantix.com — active web-development/marketing agency (existence confirmed via search snippet; page returned HTTP 403 to direct fetch). Identical name, software-adjacent sector.
- **Language:** German risk — "Krant-" is audibly close to *krank* (sick) / *kränkt*, and since German audiences do not know *kranti* = revolution, they default to that reading; "-ix" adds a comic-book tone.
- Factual note: Sanskrit *kranti* = revolution/transformation (root *kram*, "to stride") — confirmed. It appears in German yoga/spiritual sources (e.g. yoga-vidya.de Yogawiki), but is not recognized in mainstream German — the *krank* association is a real, not negligible, risk.
